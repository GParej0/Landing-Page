A modern, elegant, and **fully responsive** landing page inspired by the Mercedes-Benz brand.  
This project focuses on semantic HTML, advanced CSS styling, and modern layout techniques (Flexbox and Grid) to create a seamless experience across all devices—from desktop monitors to mobile phones.

## 📱 Fully Responsive Design

A major focus of this project was ensuring a flawless user experience on any device, regardless of screen size.

- **Fluid Layouts:** Built using CSS Grid and Flexbox to adapt naturally to different screen widths.
- **Media Queries:** Custom breakpoints ensure the navigation menu, vehicle grids, and footer restructure perfectly for tablets and smartphones.
- **Mobile-Friendly UI:** Touch-friendly buttons, readable typography, and optimized image scaling for smaller screens.

---

## ✨ Features

- **Modern & Elegant UI:** Premium design aesthetic matching the Mercedes-Benz brand identity.
- **Vehicle Showcase:** Dedicated sections for Electric and Hybrid models.
- **Interactive Elements:** Smooth hover effects, newsletter subscription form, and dealer locator section.
- **Organized Footer:** Comprehensive navigation menus and social media integration.

---

## 🛠️ Technologies Used

- **HTML5:** Semantic structuring for better accessibility.
- **CSS3:** Custom styling, transitions, and media queries.
- **Flexbox & CSS Grid:** Advanced layout management.
